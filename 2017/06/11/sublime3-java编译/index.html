sublime3下java的编译

学习java的第一步,是开发工具的选择
sublime3在偏好中找的javaC,在万能的百度上找到了这个配置文件

{

“shell_cmd”: “javac -encoding utf-8 $file_name && java $file_base_name && rm -rf $file_base_name.class”,

“working_dir”: “${project_path:${folder}}”,

“file_regex”: “^(…?):([0-9]):?([0-9]*)”,

“selector”: “source.java”,

“shell”: true,

“encoding”: “utf-8”

}

这个配置文件是可以编译java的,生成的.class也会自已删除
